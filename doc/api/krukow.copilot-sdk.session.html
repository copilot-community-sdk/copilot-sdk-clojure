send!

(send! session opts)

Send a message to the session. Returns the message ID immediately (fire-and-forget).

Options: - :prompt - The message text (required) - :attachments - Vector of {:type :path :display-name} - :mode - :enqueue (default) or :immediate

send-and-wait!

(send-and-wait! session opts)(send-and-wait! session opts timeout-ms)

Send a message and wait until the session becomes idle. Returns the final assistant message event, or nil if none received.

Options: same as send!

Additional options: - :timeout-ms - Timeout in milliseconds (default: 60000)

send-async

(send-async session opts)

Send a message and return a channel that receives events until session.idle. The channel closes after session.idle or session.error.
