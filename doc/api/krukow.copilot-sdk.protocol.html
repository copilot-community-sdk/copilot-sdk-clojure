krukow.copilot-sdk.protocol

JSON-RPC 2.0 protocol implementation over stdio/TCP streams. Uses Content-Length framing compatible with vscode-jsonrpc.

connect

(connect in out)

Create a JSON-RPC connection from input/output streams. Returns a Connection record.

connect-tcp

(connect-tcp host port)

Connect to a JSON-RPC server via TCP.

disconnect

(disconnect conn)

Close the connection.

notifications

(notifications conn)

Returns the channel that receives incoming notifications.

read-message

(read-message reader)

Read a single JSON-RPC message from reader. Returns parsed JSON map or nil on EOF.

send-notification

(send-notification conn method params)

Send a JSON-RPC notification (no response expected).

send-request

(send-request conn method params)

Send a JSON-RPC request and return a promise for the response.

send-request!

(send-request! conn method params)(send-request! conn method params timeout-ms)

Send a JSON-RPC request and block for the response. Returns result or throws on error.

set-request-handler!

(set-request-handler! conn handler)

Set handler for incoming requests. Handler is (fn method params -> channel with {:result …} or {:error …})

write-message

(write-message writer msg)

Write a JSON-RPC message to writer with Content-Length framing.
